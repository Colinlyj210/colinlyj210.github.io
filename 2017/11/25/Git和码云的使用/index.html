Git和码云的使用

发表于 2017-11-25 | 分类于 运维

Git使用

参考文档：廖雪峰的Git教程

这里主要讲一下使用SourceTree和git ssh

Sourcetree下载安装：https://www.sourcetreeapp.com/（不详细讲了）

ssh生成公钥：http://git.mydoc.io/?t=154712

码云的使用

1.在码云上建立一个项目

2.打开SourceTree，新建从URL克隆，输入码云git的地址（如果配置了ssh可以使用ssh地址就不需要密码）

编辑修改过文件使用sourcetree可以快速的推送到码云

码云的WebHooks使用

在码云上点项目然后点管理，左侧界面会有一个WebHooks选项（不知道WebHooks的自行百度），输入一个你的hook执行的网址和密码
